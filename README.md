# SkyTrack

Real-time aircraft tracking platform. See `PROJECT_PLAN.md` for full scope and phases.

## Quickstart

```bash
cp .env.example .env
docker compose up
```

Then open http://localhost:5173 — simulated aircraft should appear moving over the Marmara region.

Backend API: http://localhost:8000/api/aircraft/
